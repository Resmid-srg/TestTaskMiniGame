# TestTaskMiniGame

- Приложение реализовано на архитектуре MVVM.
 
(Реализация MVC есть в моем большом приложении для знакомств кандидатов и HR :) https://github.com/Resmid-srg/Employder)

Я решил применить архитектуру MVVM для реализации реактивного реагирования валидатора на вводимое значение при загадывании числа и реализации обработки отгадывания чисел.

- Для биндинга View и ViewModel реализован класс Dynamic

- Компьютер отгадывает число рандомными числами в урезающемся диапазоне, а не бинарным поиском (так было бы не интересно играть)

- Реализована анимация в валидаторе при загадывании числа и при отгадывании числа компьютера 

- Реализована поддержка темной темы

- Для удобства верстки повторяющийхся элементов созданы Extensions, добавлены основные цвета в Ассеты, а так же применен кусочек SwiftUI для работы режима Canvas
